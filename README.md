# Drawing Software

<img src ="https://github.com/miriamamendola/sw_eng_project_work/blob/main/src/main/resources/icon1024.png" width=14% height=14%>

#### This software allows to create simple drawings by combination of adjustable shapes.
The Drawing Software is a modern software that allows to create 
simple drawings by combination of adjustable shapes. The software,
written using Java's Swing framework and AWT libraries, provides 
essential drawing and command functionalities, thus allowing the 
user to both effectively drawn and organise the drawing.


Features:
- Shape functionalities that allow to draw both regular polygons 
and custom polygons.
- Coloring functionalities that allow, at any moment, to change both
the fill color and the stroke color of the shape.
- Selection functionality that allows to select a shape.
- Cut, Copy and Paste functionalities implemented also via predefined
keyboard shortcuts (CTRL+X, CTRL+C, CTRL+V).
- Grid functionality that allows to apply a grid to the canvas. The 
grid is also adjustable in its cell size, and has a CTRL+G keyboard
shortcut.
- Stretch and resize functionalities that allow to alter the shape
proportions or to mantain them while adjusting the size of the spape.
- Bring Forward and Backward functionalities that allow to modify the 
level at which the shape is presented. 
- Zoom functionality that allows to increase or decrease the level
of focus on a certain shape.
- Save and Load functionality that allows to save progresses regarding
a drawing. 
- Scroll functionality that allows to pan over the canvas.
- Undo functionality that allows to revert any amount of executed
actions.
- Delete functionality that allows to delete a selected shape.
